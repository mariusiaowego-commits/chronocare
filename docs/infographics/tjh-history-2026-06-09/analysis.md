# Analysis — tjh 病例汇总信息图

## Topic
父亲(tjh, 男, 64岁)过去 2 年(2024-03 ~ 2026-05)在复旦大学附属中山医院闵行院区(上海市老年医学中心)就诊的医学汇总。

## Data Type
- 量化数据为主(41 次 visits / 49 张 PDF / 7 位医生 / 6 个疾病阶段)
- 时间序列(2024-03-13 ~ 2026-05-27,共 27 个月)
- 对比数据(各医生诊断对比、初诊 vs 现状)
- 关键指标(INR 1.43~2.31 波动)

## Complexity
- 信息密度高(3 个问题、7 位医生、6 个阶段、9 个 INR 数据点、7 个对比维度)
- 需要同时呈现:医生分布、诊断一致性、病程演变、新发病

## Tone
- 温暖、关怀(给父母/家人阅读)
- 专业但不冷漠
- 避免医疗术语堆砌,关键数据点醒目标注

## Audience
- 主要读者:父亲本人(老人,65 岁左右) + 用户(成年子女,做健康管理)
- 次要:作为家庭档案长期保存

## Language
中文(简体)

## Design Instructions from User
- 选了 morandi-journal + winding-roadmap 组合
- 温暖易读、父母友好
- winding-roadmap 适合呈现"病程时间线"
- morandi 配色:奶白纸、莫兰迪绿/橘、深棕色手绘线条

## Visual Strategy
- 主体:winding-roadmap 形式呈现 6 个病程阶段(初诊→调整→稳定→检查→维持→新病情)
- 沿途里程碑:每个阶段标关键事件 + 医生 + 关键指标
- 侧边栏:bento 卡片放 7 位医生的统计 + 诊断一致性总结 + INR 趋势迷你图
- 末端:2026-05 现状总结 + 待关注事项

## Output
- 一张信息图(竖版 3:4 ~ 4:5,适合 A4 打印保存)
- 中文
- 莫兰迪手绘风 + winding-roadmap 路线图布局
