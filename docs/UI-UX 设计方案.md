## 3. 配色方案

### 3.1 主色调

```
主色 (Primary):     #2563EB (蓝)     -- 按钮、链接、主操作
次色 (Secondary):   #10B981 (绿)     -- 正常状态、成功
警告 (Warning):     #F59E0B (黄)     -- 提醒、注意
危险 (Danger):      #EF4444 (红)     -- 预警、超标、错误
信息 (Info):        #6366F1 (紫)     -- 知识库、资讯
```

### 3.2 中性色

```
背景 (Background):  #F8FAFC (浅灰白)
卡片 (Card):        #FFFFFF (白)
边框 (Border):      #E2E8F0 (淡灰)
文本主色:           #1E293B (深灰)
文本次色:           #64748B (中灰)
```

### 3.3 暗色模式

```
背景:               #0F172A
卡片:               #1E293B
边框:               #334155
文本主色:           #F1F5F9
文本次色:           #94A3B8
```

### 3.4 语义色应用

| 场景 | 颜色 | Tailwind 类 |
|------|------|-------------|
| 血糖正常 | 绿 #10B981 | `text-emerald-500` |
| 血糖偏高 | 红 #EF4444 | `text-red-500` |
| 血糖临界 | 黄 #F59E0B | `text-amber-500` |
| 血压正常 | 绿 #10B981 | `text-emerald-500` |
| 血压偏高 | 红 #EF4444 | `text-red-500` |
| 用药已服 | 绿 ✓ | `bg-emerald-100 text-emerald-700` |
| 用药漏服 | 红 ✗ | `bg-red-100 text-red-700` |
| 预警卡片 | 红边框 | `border-red-500 bg-red-50` |

## 4. 组件设计规范

### 4.1 卡片 (Card)

```html
<div class="bg-white dark:bg-slate-800 rounded-xl shadow-sm border border-slate-200 
            dark:border-slate-700 p-6">
  <h3 class="text-lg font-semibold text-slate-900 dark:text-slate-100">标题</h3>
  <p class="text-slate-600 dark:text-slate-400 mt-2">内容</p>
</div>
```

### 4.2 预警卡片 (Alert Card)

```html
<div class="flex items-start gap-3 p-4 rounded-lg border-l-4 
            border-red-500 bg-red-50 dark:bg-red-950">
  <span class="text-red-500 text-xl">🔴</span>
  <div>
    <p class="font-medium text-red-800 dark:text-red-200">血糖偏高</p>
    <p class="text-sm text-red-600 dark:text-red-300">空腹 8.2 mmol/L，超过阈值 7.0</p>
  </div>
</div>
```

### 4.3 数据表格

```html
<table class="w-full text-sm">
  <thead class="bg-slate-50 dark:bg-slate-800">
    <tr>
      <th class="px-4 py-3 text-left font-medium text-slate-500">日期</th>
      <th class="px-4 py-3 text-left font-medium text-slate-500">数值</th>
      <th class="px-4 py-3 text-left font-medium text-slate-500">状态</th>
    </tr>
  </thead>
  <tbody class="divide-y divide-slate-200">
    <tr class="hover:bg-slate-50">
      <td class="px-4 py-3">05-02</td>
      <td class="px-4 py-3 text-red-500 font-medium">8.2</td>
      <td class="px-4 py-3"><span class="badge-danger">偏高</span></td>
    </tr>
  </tbody>
</table>
```

### 4.4 快速输入表单

- 大号输入框，方便手机点击
- 数字输入用 `inputmode="decimal"` 弹出数字键盘
- 日期用原生 `datetime-local` 输入
- 提交按钮大且醒目

### 4.5 HTMX 交互模式

```html
<!-- 无需刷新的表单提交 -->
<form hx-post="/api/blood-sugar" hx-target="#record-list" hx-swap="afterbegin">
  <input name="value" type="number" step="0.1" inputmode="decimal">
  <button type="submit" class="btn-primary">保存</button>
</form>

<!-- 自动加载的统计卡片 -->
<div hx-get="/api/blood-sugar/stats?person_id=1&period=7d" 
     hx-trigger="load, every 5m" hx-swap="innerHTML">
  加载中...
</div>
```

## 5. 移动端适配策略

### 5.1 响应式断点

```
sm:  640px   -- 手机横屏
md:  768px   -- 平板
lg:  1024px  -- 桌面
xl:  1280px  -- 大桌面
```

### 5.2 移动端布局变化

| 桌面端 | 移动端 |
|--------|--------|
| 侧边栏 + 主内容 | 底部 Tab Bar + 全宽内容 |
| 多列统计卡片 | 单列堆叠 |
| 表格横向滚动 | 卡片列表替代表格 |
| 鼠标悬停操作 | 左滑显示操作按钮 |

## 6. 数据可视化方案 (Chart.js)

### 6.1 血糖趋势图

- **类型**: 折线图 (Line Chart)
- **X 轴**: 日期时间
- **Y 轴**: 血糖值 (mmol/L)
- **参考线**: 水平虚线标注正常范围 (3.9-6.1)
- **阈值线**: 红色虚线标注预警阈值 (7.0/10.0)
- **数据点**: 正常绿色，超标红色
- **交互**: hover 显示详细信息

### 6.2 血压趋势图

- **类型**: 双线折线图
- **线条**: 收缩压(红) + 舒张压(蓝)
- **参考带**: 正常范围半透明背景色
- **心率**: 可选叠加心率折线

### 6.3 用药依从性图

- **类型**: 热力日历图 (GitHub contribution 风格)
- **颜色**: 全服(绿) → 部分(黄) → 漏服(红) → 未记录(灰)
- **周期**: 月视图

### 6.4 统计仪表盘卡片

- **类型**: 数字卡片 + 迷你趋势线 (Sparkline)
- **内容**: 平均值、最高/最低、达标率
- **趋势**: 小型折线显示近7天走向
