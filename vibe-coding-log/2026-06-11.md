## 关键学习
1. HTMX innerHTML 不执行 script 标签 — JS 必须放在页面级
2. `hermes status` 显示的 model 不是 image gen 实际用的模型 — 需要真实调用验证
3. GPT-4o 能处理复杂信息图 — 问题在 prompt 结构而非模型能力
4. 诊断归一化很重要 — "二尖(瓣)关闭不全" vs "二尖瓣关闭不全" 会导致共识诊断为空
